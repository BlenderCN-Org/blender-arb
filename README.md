Accurate Render Border
====================================================================================
Blender - setting render border with accurate values

====================================================================================
Installation

User Preferences - Addons - Install from file
    
    __init__.py

====================================================================================
Usage

In T-bar set values on ARB tab and press button "Set Accurate Values"
